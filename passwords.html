Password Strength

What is a strong password?
  • A strong password contains a combination of both uppercase and lowercase letters, digits, and symbols.
  • It has at least 8 characters.
  • It doesn't contain common words or phrases.
Why do we need strong passwords?
  • A brute force attack is a method of cracking passwords that tries all possible character combinations to find a password.
    • A total of 2.1 · 1011 possibilities exist in a password consisting of 8 lowercase letters.
    • A total of 2.8 · 1012 possibilities exist in a password consisting of 8 mixed-case letters.
    • In general, the more types of characters and the more characters you have in your password, the better.
    • Time is the only limiting factor in a brute force attack.
    • Any password can be found using this method, given time.
    • This type of attack exploits those with short passwords, usually under 8 characters.
  • A dictionary attack is a method that compares your password to a list of commonly used or previously cracked passwords.
    • No matter how long or complex a password is, it will be found if it is in the list.
    • Unlike a brute force attack, dictionary attacks are not limited by time.
    • The limitation is whether or not the password is in the list.
    • This type of attack exploits those who use the same password for every log-in.

Default Passwords

What are default passwords?
  • Manufacturers provide default usernames and passwords to make the initial setup of a device simpler.
  • Default credentials are also used whenever a device is reset back to factory settings.
  • Default passwords are usually simple, and not unique to each individual device.
  • Default passwords can be found in the instruction manuals, or on the device itself. There are many websites that list the default credentials for many devices.
  • Attackers can easily obtain default passwords and identify internet-connected target systems.

Here is one of many websites that list default passwords:  http://www.defaultpassword.com/

What should you do?

Always change default passwords as soon as possible on any internet-connected device, absolutely before you connect to the internet.
